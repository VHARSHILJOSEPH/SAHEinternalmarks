# VRSEC Internal Marks Calculator

A web tool to help students compute internal marks based on tests, assignments, and exam scores.

## Features

- Input your Class Test, Sessional, and Home Assignment marks.
- Calculates total internal marks (out of 40).
- Displays grade requirement table for end-exam scores.

## How to Use

1. Open `index.html` in your browser (download or clone the repo).
2. Enter your marks and click **Calculate Internal Marks**.
3. View your calculated results and grade requirements for the end-exam.

## Example

```
Class Test 1: 7
Class Test 2: 10
Sessional 1: 14
Sessional 2: 18
Home Assignment: 10
```

**Result Example:**
- Class Test Marks: 9
- Sessional Marks: 17
- Home Assignment: 10
- **Total Internal Marks (out of 40): 36**
- Table shows how many marks you need in the end exam for each grade.
